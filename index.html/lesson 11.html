THE POSITION PROPERTY

The Position Property is used to specify the positioning method of an element. The top, left, bottom, and right properties determines the final location of positioned element.

Position Property's value(s).

  1. relative
  2. absolute
  3. fixed
  4. static
position relative
Position relative property is used to position an element according to the normal flow of the document, and then offset relative to itself based on the values of top, left, bottom, and right. The offset does not affect the position of any other elements.
position absolute
Position absolute is used to remove an element from the normal document flow, and no space is created for the element in the page layout. The element will be positioned relatve to its closest ancestor or to the inital containing block.
  • Its final position is determined by the values of top, left, bottom, and right
position fixed
an element with positioned: fixed; is positioned relative to the viewport, which means it always stays in the same place even if the page is scrolled. A fixed element does not leave a gap in the edge where it would normally have been located.
  • Its final position is determined by the values of top, left, bottom, and right
position static
static is the default position for HTML elements. elements with position: static; are positioned based on the normal flow of the page. they are not affected by the top, right, left, bottom properties
  • Its final position is determined by the values of top, left, bottom, and right
