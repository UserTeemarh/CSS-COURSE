THE POSITION PROPERTY

The position property is used to specify how an element is positioned within its parent element or the viewport. It is used to determine where an element is displayed on the page.

TYPES OF POSITION PROPERTY'S VALUES

POSITION PROPERTY'S VALUES AND DESCRIPTION
POSITION PROPERTY'S VALUES DESCRIPTION
static The static value is the default positioning method for elements. It makes an element follow the normal flow of the document and it appears in the order it's written in the HTML and doesn't undergo any special positioning behavior.
relative The relative value makes the element positioned to its normal position in the document flow, but the element can be adjusted by using relative properties
RELATIVE VALUE'S PROPERTY
top The top property specifies the distance the element is moved down from its normal position in the document flow. It makes the element remains in the document flow, and its position is adjusted relative to where it would be normally be placed.
The value can specified in numerical values with units
left The left property specifies the distance the element is moved horizontally from its normal position in the document. This allows the element to be adjusted horizontally to where it would be normally be placed
The value can specified in numerical values with units
bottom The bottom property specifies the distance the elemetn is moved up from its normal position in the document flow. This allows the element's vertical position to be adjusted relative to where it would normally be placed.
The value can specified in numerical values with units
right The right property specifies the distance the element is moved horizontally from its normal postions in the document flow.
The value can specified in numerical values with units
Absolute: The absolute value is used to remove element from the normal document flow and position it relative to its nearest positioned ancestor (or the initial containing block if no positioned ancestor is found). This implies that the element is taken out of the flow of the document, and its position is calculated based on its closest ancestor with position value
ABSOLUTE PROPERTY'S VALUE
top The top property is used to specify the distance element is positioned from the top edge of its containing block
The value can specified in numerical values with units
left The left property is used to specify the distance the element is positined from the left edge of its containing block.
The value can specified in numerical values with units
bottom The bottom property is used to move the element up from the bottom edge of its containing block by the specified distance.
The value can specified in numerical values with units
right The right property is used to specify the distance the element is positined from the right edge of its containing block.
The value can specified in numerical values with units
fixed The fixed value is used to positioned element to be relative to the viewport (the browser window), and makes it remain fixed in its position even when the page is scrolled.
FIXED VALUE'S PROPERTY
top The top property specifies the distance the element is positioned from the top edge of the viewport.
The value can specified in numerical values with units
left The left property is used to specify the distance the element is positined from the left edge of its containing block.
The value can specified in numerical values with units
bottom The bottom property is used to move the element up from the bottom edge of its containing block by the specified distance.
The value can specified in numerical values with units
right The right property is used to specify the distance the element is positined from the right edge of its containing block.
The value can specified in numerical values with units
sticky The sticky value positions based on the user's scroll position. It behaves like relative positioning until the element reaches a specified scroll position, then it "sticks" in place like fixed positioning.
STICKY VALUE'S PROPERTY
top The top property determines the offset from the top edge of the containing block at which the element becomes sticky.
The value can specified in numerical values with units
left The left property determines the offset from the left edge of the containing block at which the element becomes sticky
The value can specified in numerical values with units
bottom The bottom property determines the offset from the bottom edge of the containing block at which the element becomes sticky
The value can specified in numerical values with units
right The right property determines the offset from the right edge of the containing block at which the element becomes sticky
The value can specified in numerical values with units

z-index property is crucial to positioning property because its the type of property that controls the order of positioned elements that overlap within the same stacking context.

How It Works:

  • Elements with higher z-index value are stacked in front of elements with lower z-index values within the stacking context.
  • If two elements have the z-index, the one that comes later in the HTML document order will appear on top.
  • The z-index property only works on elements with a position value of absolute, relative, fixed, or sticky, It has no effect on elements with static value (the default value).
