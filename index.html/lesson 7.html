CSS BOX MODEL

The css box model is a fundamental concept that defines the layout and structure of elements on a webapage. It collectively determine the dimensions and spacing of an element.

COMPONENTS OF CSS BOX MODEL

1. CONTENT

The content refers to the actual content of the HTML element, such as text, images, or other media. It is the innermost part of box model. The size of the content area can be specified using CSS properties like width and height.

2. BORDER

The border refers to the area that surrounds the content and padding of an element. It defines the visible boundary of the element and seperates it from adjacent elements.

BORDER PROPERTIES AN DESCRIPTION
BORDER PROPERTY DESCRIPTION VALUE(s)
border-width The border-width property is used to specify the width of the border around an element. It can be set for individual sides (top, right, bottom,left).
  • length values with units.
  • thin
  • medium
  • thick
border-style The border-style property defines the style of the border around an element. It specifies how the border should be displayed.
  • none
  • solid
  • dashed
  • dotted
  • double
  • groove
  • ridge
  • inset
  • outset
border-color The border-color is used to specify the color of the border around an element. It accepts various color values.
  • named colors
  • hexadecimal
  • RGB/RGBA
  • HSL/HSLA
  • transparent
border-radius The border-radius is used to to create rounded corners for an element's border. It is used to control the curvature of each corner individually or set the same radius for all corners
  • length values
border The border is a shorthand property that allows the specification of all the individual border properties (border-width, border-style, and border-color) in one declaration.
  • border-width, border-style, and border-color

3. MARGIN

The margin property is used to create space around an element's border. It defines the space between the element's border and adjacent element.

INDIVIDUAL MARGIN PROPERTIES AND DESCRIPTION
MARGIN PROPERTY DESCRIPTION
margin-top Specifies the margin for the top side of the element
margin-right Specifies the margin for the right side of the element
margin-bottom Specifies the margin for the bottom side of the element
margin-left Specifies the margin for the left side of the element
margin Sets the margin for all four sides of an element

4. PADDING

The padding property is used to create space around the content inside an element's border. It defines the distance between the element's content area and its border.

INDIVIDUAL PADDING PROPERTIES AND DESCRIPTION
PADDING PROPERTY DESCRIPTION
padding-top Specifies the padding for the top side of the element
padding-right Specifies the padding for the right side of the element
padding-bottom Specifies the padding for the bottom side of the element
padding-left Specifies the padding for the bottom side of the element
padding Sets the padding for all four sides of an element
