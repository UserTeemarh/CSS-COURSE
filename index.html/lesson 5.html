CSS BACKGROUND PROPERTIES

The CSS background properties are used to define the background effect for elements (what paints underneath the content in that element). The Background properties are used to control the appearance of the background of an element.

TYPES OF BACKGROUND PROPERTIES IN CSS.

BACKGROUND PROPERTIES AND DESCRIPTIONS
BACKGROUND PROPERTY DESCRIPTION BACKGROUND VALUE(s)
background-color The background-color property is used to specify the background color of an element. It sets the color that appears behind the content and padding areas of an element, and it accepts a variety of color values.
  • Named Color
  • hsl/hsla
  • rgb/rgba
  • hex
background-image The background-image property specifies one or more background images for an element. It allows the specification of setting the image that appears behind the content and padding areas of element. Multiple images can be used for an element by comma-seperating each of the images.
  • url()
background-repeat The background-repeat property is used to specify how a background image should repeat both horizontally and vertically within an element's background area. It determines whether the background image should repeat, repeat only horizontally, or not repeat at all.
  • repeat
  • repeat-x
  • repeat-y
  • no-repeat
background-position The background-position is used to specify the starting position of a background image within an element's background area. It determines where the background image will be placed relative to the element's padding box.
  • top
  • left
  • right
  • center
background-size The background-size is used to specify the size of a background image within an element's background area. It determines how the background image should be scaled or sized to fit the element
  • auto
  • cover
  • contain
  • length values (pixels, percentages)
background-attachment The background-attachment property is used to specify whether a background image should scroll with the content or remain fixed as the content is scrolled. It determines whether the background image is fixed in relation to the viewport or scrolls along with the element's content
  • scroll
  • fixed
  • local
background-origin The background-origin property is used to determine the postioning area of the background image and background-color within an element's box. It specifies where the background images or colors originate from.
  • padding-box
  • border-box
  • content-box
background-blend-mode The background-blend-mode property is used to specify how the background image of an element should blend with its background color or other background images. It determines how the colors of the background layers should be blended together.
  • darken
  • lighten
  • screen
  • multiply
  • overlay
  • exclusion
  • color-burn
  • color-dodge
background-clip The background-clip is used to specify how the background of an element should be clipped in relation to the element's border or its padding box. It determines the area within which the background of an element is visible.
  • text
  • border-box
  • content-box
  • content-box
