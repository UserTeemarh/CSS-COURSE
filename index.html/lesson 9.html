FLEXBOX

Flexbox, short for "flexible Box Layout". is a layout model designed to provide a more efficient way to arrange and align elements within a container, especially when dealing with dynamic or unknown sizes of the elements. It aims to provide a more efficient and predictable way to distribute space among items in a container.

Flexbox introduces the concept of Flex containers and Flex items. The container element becomes a flex container, and its child elements becomes flex items

Flexbox provides a powerful and intuitive way to create flexible and responsive layouts, reducing the reliance on floats, positioning, and complex CSS hacks. It simplifies the process of creating complex layouts.

TYPES OF FLEXBOX PROPERTIES

FLEXBOX PROPERTIES AND DESCRIPTION
FLEXBOX PROPERTY DESCRIPTION VALUE(s)
display: It is used to define an element as a flex container. It's the fundamental property for using flexbox layout.
  • flex
  • inline-flex
flex-direction It determines the direction of the main axis within a flex container. This main axis defines the direction in which flex items are laid out.
  • row
  • row-reverse
  • column
  • column-reverse
flex-wrap It determines whether flex items are forced into a single line or can be wrapped onto multiple lines within the flex container when there isn't enough space.
  • wrap
  • nowrap
  • wrap-reverse
flex-flow It is a shorthand property that combines the flex-direction and flex-wrap properties in a single declaration.
  • flex-direction, flex-wrap
justify-content It is used to align flex items alogn the main axis of the flex container. It controls the distribution of space between and around flex items when they don't take up all the available space on the main axis.
  • flex-start
  • flex-end
  • center
  • space-between
  • space-around
  • space-evenly
align-items It is used to align flex items along the cross axis of the flex container. It controls how flex items are aligned vertically (for a flex-direction: row layout) or horizontally (for a flex-direction: column layout).
  • stretch
  • flex-start
  • flex-end
  • center
  • baseline
align-content It is used to align a flex container's lines within the flex container when there's extra space in the cross-axis. It only has an effect when there are multiple lines of flex items within the flex container.
  • stretch
  • flex-start
  • flex-end
  • center
  • space-between
  • space-around
flex-grow It is used to specify how much a flex item should grow relative to the other flex items within the same flex container when there is extra space available along the main axis. It determines the proportion of the available space that a flex item should occupy.
  • number
flex-shrink It is used to specify how a flex item shrink relative to the other flex items within the same flex container when there is not enough space along the main axis. It determines the ability of a flex item to shrink in order to fit into the available space.
  • number
flex-basis It is used to specify the initial main size of a flex item before free space is distributed. It sets the initial size of a flex item along the main axis before considering the glex-grow and flex-shrink properties
  • Number with units
flex: The flex property is used to combine flex-grow, flex-shrink, and flex-basis in a single declaration.
  • flex-grow, flex-shrink, flex-basis
