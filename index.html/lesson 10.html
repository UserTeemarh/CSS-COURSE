THE GRID

A grid is a two-dimensional layout system that is used to create complex web page layouts with rows and columns. It's defined by a set of rows and columns, where content can be placed within individual cells or across multiple cells.

The grid layout module provides a powerful way to arrange elements on a web page, offering more flexibility and control compared to traditional layouts methods like floats and positioning.

TYPES OF GRID PROPERTIES

GRID PROPERTIES AND DESCRIPTION
GRID PROPERTY DESCRIPTION GRID VALUE(s)
display The display property is used to define a container as a grid container, enabling grid layout functionality.
  • grid
grid-template-columns The grid-template-columns property is used to define the number and size of columns in a grid container. It allows the specificity of the width of each column individually or using various units.
  • fixed values (px, pt, in)
  • percentages value (%)
  • Fractional units (fr)
  • Auto
  • fit-content
grid-template-rows The grid-template-rows property is used to define the size of each row in the grid. It allows the specificity of the height of each row individually or collectively using a combination of length values
  • fixed values (px, pt, in)
  • percentages value (%)
  • Fractional units (fr)
  • Auto
  • fit-content
grid-gap or gap The grid-gap or gap is used to specify the size of the gap between grid items in a grid container. It sets the gap both horizontally or vertically.
  • Single value with unit (px/pt/in)
  • Two values with units (px/pt/in)
grid-row-start The grid-row-start property specifies the line on which the grid item's row starts. It determines the starting position of the grid item along the grid's rows.
  • Numerical values with no unit
  • Named Lines: Uses custom names for the grid lines, if defined.
  • Span: Specifies how many grid tracks the item should span.
grid-column-start The grid-column-start property specifies the line on which the grid item's column starts. It determines the starting position of the grid item along the grid's columns.
  • Numerical values with no unit
  • Named Lines: Uses custom names for the grid lines, if defined.
  • Span: Specifies how many grid tracks the item should span.
grid-row-end The grid-row-end property specifies the line on which the grid item's column ends. It determines the ending position of the grid item along the grid's columns.
  • Numerical values with no unit
  • Named Lines: Uses custom names for the grid lines, if defined.
  • Span: Specifies how many grid tracks the item should span.
grid-column-end The grid-column-end property specifies the line on which the grid item's column ends. It determines the ending position of the grid item along the grid's columns.
  • Numerical values with no unit
  • Named Lines: Uses custom names for the grid lines, if defined.
  • Span: Specifies how many grid tracks the item should span.
grid-column The grid-column property is used to specify the grid lines on which a grid item will be placed, both its starting and ending postions along the grid's columns.
  • Line numbers. e.g (1/3)
  • Span(It specifies how many grid tracks the item should span).
  • Auto: It is used to let the grid layout algorithm determine the placement.
grid-row The grid-row property is used to specify the grid lines on which a grid item will be placed, both its starting and ending positions along the grid's rows.
  • Line numbers. e.g (1/3)
  • Span(It specifies how many grid tracks the item should span).
  • Auto: It is used to let the grid layout algorithm determine the placement.
grid-area The grid-area property is a shorthand property that allows the use of (grid-row-start, grid-column-start, grid-row-end, grid-column-end) in a single declaration. It assigns a grid item to a named grid area.
  • grid-row-start / grid-column-start / grid-row-end / grid-column-end
justify-content The justify-content property is used to align the grid along the inline (row) axis within the grid container. It determines how grid items are distributed horizontally where there is extra space in the grid container.
  • start
  • end
  • center
  • stretch
  • space-between
  • space-around
align-content The align-content property is used to align the grid along the block (column) axis within the grid container when there's extra space available. It doesn't have any effect if there's only one row of grid items.
  • start
  • end
  • center
  • stretch
  • space-between
  • space-around
justify-items The justify-items property is used to align grid items along the inline (row) axis within their grid cell. It affects how grid items are positioned inside their respective grid areas vertically.
  • start
  • end
  • center
  • stretch
align-items The align-items property is used to align items along the block (column) axis within their grid cells. It affects how grid items are positioned inside their respective grid areas horizontally.
  • start
  • end
  • center
  • stretch
