THE ANIMATION PROPERTY

The Animation property is used to apply animations to elements. It combines various animation-related properties into a shorthand.

Animation involve the use of the @keyframes to define the sequence of style for an element at different points in time.

How to make an animation

  • select an HTML element to animate
  • a CSS rule which binds the animation to the element
  • a group of @keyframes that defines the styles at the start and the end of the animation.

Types of Animation Properties

The animation property is a shorthand property that combines several individual animation-related properties. The components of the animation property are as follow:

  • Animation-name
  • Animation-duration
  • Animation-timing-functions
  • Animation-iteration-count
  • Animation-delay
  • Animation-fill-mode
  • Animation-play-state
Animation-name
The Animation-name property is used to specify the name of the @keyframes rule that defines the animation behavior.
Animation-duration
The Animation-duration property is used to specify how long the animation cycle should take. The time is specified in seconds or milliseconds, which means that the animation occurs instantenously.
Animation-timing-function
The Animation-timing-function property is used to define the pace of an animation. It specifies how the intermediate values of the CSS properties being animated are calculated over the duration of the animation.

Animation-timing function values are as follow:

  • ease:(default)  slow at the beginning, fast in the middle, slow at the end
  • linear:   constant speed throughout the animation
  • ease-in:   starts slowly and accelerate
  • ease-out:   starts quickly and decelerates
  • ease-in-out:   starts slowly, accelerates in the middle, and decelerates at the end.
Animation-iteration-count
The Animation-iteration-count property in specifies the number of times an animation cycle should be played. It takes a numberic value or the keyword infinte for endless repetition
Animation-delay
The Animation-delay property specifies the amount of time to walt from applying the animation to an element before beginning to perform the animation.
Animation-fill-mode
The Animation-fill-mode property controls the style of an animated element outside of its execution

Animation-fill-mode values are as follow:

  • none:(default)   The animations styles are not changed before or after the animation.
  • forwards:   The element retains the styles of the last keyframes after the animation finishe
  • backwards:    The element is styled with the values of the first keyframe before the animation starts
  • both:    combines the effect of forwards and backwards, meaning the element is styled with the values of the first keyframe before the animation starts and retains the styles of the last keyframe after the animation finishes
Animation-play-state
The Animation-play-state property is used to pause or play an animation

It take two values

  • running
  • pause
