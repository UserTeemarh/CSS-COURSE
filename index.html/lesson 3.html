4.    PSEUDO SELECTOR

Pseudo Selector is a keyword added to a selector that specifies a special state of the selected element(s). It is used to add styles to selectors, but only when those selectors meet certain conditions.

TYPES OF PSEUDO SELECTOR

Pseudo-class Selectors

A pseudo class is expressed by adding a colon (:) after a selector.

TYPES OF CSS PSEUDO-CLASS SELECTORS
pseudo-class selector Description
:active Represents an element (such as a button) that is being activated by the user.
when using a mouse "activation" typically starts when the user presses down the primary mouse button
:hover It matches when the user interacts with an element with a pointing device, and does not necessarily activate it. it is generally triggered when the user hovers over an element with the cursor (mouse pointer)
:link It represents an element that has not yet been visited. it matches every unvisited < a > element that has an href attribute
:visited It applies once the link has been visited by the user, for privacy reasons, the styles that can be modified using the selector are very limited
:first-child It represents the first element among a group of sibling elements. It is used to target the first-child element of its parent for styling
:last-child It represents the last element among a group of sibling elements. It is used to target the last-child element of its parent for styling
:nth-child It matches elements based on the indexes of the elements in the child list of their parents.
:empty It selects any element with no content, whether text or child elements.
The selectors looks for element with no child nodes text-content, or whitespace characters. The :empty does not consider an element that has spaces, tabs or line breaks to be empty.
:disabled It represents any disabled element. An element is disabled if it can't be activated (selected, clicked on, typed into.) or accepts focus. The element also has an enabled state, in which it can be activated or accept focus.
:valid It represents any <input> or other <form> element whose contents validate successfully.  This allows to easily make valid fields adopt an appearance that helps the user confirm that their data is formatted properly.
:invalid It is used to select every form element which does not validate according to the elements. this selectors works only for some form elements with limitation such as input elements with min and max and max attributes, emails fields with legal email, or email fields with a numeric value. etc.
:checked It is used to select and style radio buttons or checkbox that are checked.

Pseudo-element Selectors

A pseudo element is expressed by adding a double colon (::) after a selector

PSEUDO-ELEMENT SELECTOR
Pseudo-element Selector Description
::before It is used to selects and styles content before the actual content in an element.
::after It is used to selects and styles content after the actual content in an element.
::first-line It is used to add style to the first line of the specified selector.
::first-letter It is used to add style to the first letter of the specified selector.
::selection It is used to selects and styles the portion that's selected by the user (such highlighting).
::marker It selects the marker box of list item, which typically contains a bullet or number.
