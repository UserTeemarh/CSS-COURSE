4.    PSEUDO SELECTOR

Pseudo Selector is a keyword added to a selector that specifies a special state of the selected element(s). It is used to add styles to selectors, but only when those selectors meet certain conditions.

TYPES OF PSEUDO SELECTOR

Pseudo-class Selectors

A pseudo-class is used to define a special state of an element that is not represented by simple HTML attributes. Pseudo-class are preceded by a colon ":" and target specific elements based on their interaction or element relationships.

TYPES OF CSS PSEUDO-CLASS SELECTORS
pseudo-class selector Description
:active It is used to select and style an element while it is being activated or clicked by the user
:hover It is used to select and style an element when it is hovered over by the cursor
:link It is used to select and style links that have not been visited by the user. It targets the default state of links.
:visited It is used to select and style links taht have been visited by the user. It targets the state of links that have been already been clicked on.
:first-child It represents the first element among a group of sibling elements. It is used to select and style the first-child element of its parent for styling
:last-child It represents the last element among a group of sibling elements. It is used to select and style the last-child element of its parent for styling
:nth-child It is used to select and style elements based on their position within their parent element. It allows the element to be targeted based on a specific formula.
:empty It is used to select and style elements that have no children, including text nodes and whitespace.
:disabled It is used to select and style form elements that are disabled. This includes input fields, buttons, and other form element that have the disabled attribute.
:valid It is used to select and style form elements that have passed validation.
:invalid It is used to select and style form elements that have failed validation.
:checked It is used to select and style form elements that are checked or selected. It is commonly used with checkboxes, radio buttons, and select dropdowns.

Pseudo-element Selectors

A pseudo-element selector is used to style a specific part of an element's content. Unlike pseudo-class, which targets states of elements or relationships between element, pseudo-element targets parts of elements themselves. Pseudo-elements are preceded by a double colon ": :"

PSEUDO-ELEMENT SELECTOR
Pseudo-element Selector Description
::before It is used to insert content before the content of the selected element. It allows the insertion of texts, images before an element's actual content.
::after It is used to insert content after the content of the selected element. It allows the insertion of texts, images before an element's actual content.
::first-line It is used to select and style the first line of text within a block-level element. It allows styles to be specifically applied to the first line of text within an element.
::first-letter It is used to select and style the first letter of text within a block-level element. It allows styles to be specifically applied to the first letter of text within an element.
::selection It is used to style the portion of a document that has been highlighted by the user.
::marker It selects the marker box of list item, which typically contains a bullet or number.
