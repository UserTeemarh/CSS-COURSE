COLOR PROPERTY

The color property is used to set the text color of an element. It allows the customization of text color to achieve the decimal visual appearance for the webpage.

Types of CSS Color Code

Types of CSS Color Code Description Color Illustration
Named Color Color names are value (name) that represent a color. There are around 147 named colors recognized by browsers. Brown
red
yellow etc.
HSL HSL stands for hue, saturation, and lightness. It is another way of specifying color for text (and anything else that takes color) in CSS. hsl(23, 50% 40%)
  • Hue represents the color wheel in 360°. So, 0° is red, 120° is green and 240° is blue.
  • Saturation is the amount of gray in the color, expressed as a percentage. 0% is the shade of gray and 100% is the color itself.
  • Lightness is the amount of darkness and lightness in the color expressed as a percentage. 0% is black and 100% is white.
HSLA HSLA color value is an extension of HSL color values with an alpha channel - which specifies the opacity of an element. The extra ‘a’ stands for alpha, which allows the specification of the color opacity. It takes a value from 0.0 to 1.0 hsla(23, 50%, 40%, 0.5)
HEX Hex values are used to represent colors with a total of 6 characters. It is denoted with (#) symbol, then any number from 0 to 9, and any letter from A to F. #f00ff
The first 2 values stand for red, the next two stand for green, and the last 2 represent blue. With hex values, there's no limitation to usage of the shades of colors.
RGB RGB stands for red, green, and blue. With RGB colors, It can be specified in terms of how much red, green, blue required. All three are expressed with numbers between 0 and 255. rgb(255, 255, 234)

rgba(255, 255, 234, 0.5)
There is a type of RGB called rgba. The extra ‘a’ stands for alpha, which allows the specification of the color opacity. It takes a value from 0.0 to 1.0

Which Color Code is the Best to use?

  • When named color is used. The use of different shades of colors is limited because the browser recognized only 147 predefined colors.
  • Hexadecimal colors are very dynamic because it doesn't have a limitation. With hex colors, various shades of color can be used.
  • RGB colors are as dynamic as hex colors. Apart from being able to specify how much red, green, and blue to be used from 0 to 255, the transparency of the color can be set with the extra alpha value.
  • HSL is the most dynamic of all. Colors can be specified in degrees from 0 to 360 within the color wheel, set how saturated and dark it should look in percentages, and can also set an opacity from 0.0 to 1.0
