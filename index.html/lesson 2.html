CSS SELECTOR

A css selector is a pattern or expression that specifies which HTML elements should be selected and styled on a webpage. It allows specific elements to be targeted or groups of elements to apply styles.

CSS selectors play a key role in the cascading nature of styles, where rules are applied in a specific order, affecting the appearance of elements on the page.

TYPES FOR CSS SELECTORS

1.   BASIC CSS SELECTOR

BASIC CSS SELECTOR
BASIC TYPE OF CSS SELECTOR DESCRIPTION
Universal Selector the universal selector is a wildcard character that selects all HTML elements on a webpage. It is used to apply styles globally to every element. universal selector is denoted by the asterik (*) .
Element Selector an element selector is a basic selector that targets HTML elements based on their tag name. It selects and styles all instance of a particular HTML element.
ID Selector an ID selector is used to target a specific HTML element based on its unique ID attribute. The ID selector is denoted by a hash (#) followed by the value.
The id of an element is unique within a page, so the id selector is used to select one unique element!
Class Selector a class selectors is used to target HTML elements based on their class attribute, class selectors are denoted by a dot (.) followed by the class name. It selects all elements with that class attribute so that unique CSS declarations can be applied to those specific elements without affecting other elements on the page.
class selector can be used multiple times to select elements on a page unlike id selectors.

2.    COMBINATION SELECTORS

A combination selector is a selector that combines multiple basic selectors to target specific elements or relationships between elements in a more precise way.

COMBINATION SELECTORS and DESCRIPTION
COMBINATION TYPE OF CSS SELECTORS DESCRIPTION
Descendant Selector The descendant selector is a way to select elements that are located somewhere underneath other elements. It involves placing a space between the selectors.
Child Selector The child selector denoted with (>) symbol is used to select elements that are direct children of a specified parent element. It only targets elements that are one level deep in the HTML hierachy.
Adjacent Sibling Selector The adjacent sibling selector denoted with (+) is used to select an element that is immediately preceded by a specified element. It targets an element that shares the same parent and comes directly after the specified element in the HTMl structure.
General Sibling Selector The general sibling selector denoted with a tilde (~) is used to select element and share the same parent. It targets elements that come after the specified element, not necessarily immediately after.

3.    ATTRIBUTE SELECTOR

Attributes Selectors are patterns that allows the selection of HTML elements based on the presence or value of their attributes. These selectors contribute to the flexibility and specificity of style rules.

ATTRIBUTE SELECTOR and DESCRIPTION
ATTRIBUTE TYPE OF CSS SELECTOR DESCRIPTION
[attr] Represents elements with an attribute name of attr.
[attr=value] Represents elements with an attribute name of attr whose value is exactly value.
[attr~=value] Represents elements with an attribute name of attr whose value is a whitespace-separated list of words, one of which is exactly value.
[attr|=value] Represents elements with an attribute name of attr whose value can be exactly value or can begin with value immediately followed by a hyphen, -.
[attr^=value] Represents elements with an attribute name of attr whose value is prefixed (preceded) by value.
[attr$=value] Represents elements with an attribute name of attr whose value is suffixed (followed) by value.
[attr*=value] Represents elements with an attribute name of attr whose value contains at least one occurrence of value within the string.
