CSS PROPERTIES

CSS properties is an attributes of an HTML element that defines how the element should be styled or displayed on a webpage. It controls the appearance and layout of the web page.

FONT PROPERTIES

Font properties are used to control various aspects of text rendering.

FONT PROPERTIES AND DESCRIPTION
FONT PROPERTY DESCRIPTION VALUE(s)
font-size: The font-size property defines the size of the font used for text content within an element. It can be set to various units. Increasing the value makes the text larger, while decreasing it makes the text smaller.
UNITS
  • pixels (px)
  • points (pt)
  • percentages (%)
font-style The font-style is used to specify the style of the font within an element.
  • normal
  • italic
  • oblique
font-family The font-family is used to specify the typeface or font family that will be applied to text content within an element. It allows developers to define a prioritized list of font family names, seperated by commas, in case the preffered is not available on the user's system. font name
font-weight The font-weight property is used to specify the thickness or boldness of the font. It accepts numeric values or predefined keywords.
  • normal
  • bold
  • lighter
  • bolder
  • Numeric values ranging from 100 to 900.
    • 100 to 500 represents lighter weights
    • 600 to 900 represents bolder weights.
font-variant The font-variant property is used to control the use of small caps or normal font variants.
  • normal
  • small-caps
text-transform The text-transform property is used to specify how the text content within an element should be transformed in terms of capitalization.
  • none
  • uppercase
  • lowercase
  • capitalize
text-decoration The text-decoration property is used to add visual effects to text. It is commonly used to enchance the visual appearance of links, emphasize certain text, or denote formatting within an element.
  • none
  • underline
  • overflow
  • line-through
text-shadow The text-shadow property adds a shadow effect to the text. It allows the horizontal and vertical offset to be specified as well as the blur radius and color.
  • h-shadow, v-shadow, blur-radius, color
white-space The white-space property controls how white spaces, line breaks, and tabs are processed and rendered within an element's content. It can be used to control text-wrapping, preserve, formatting, and manage white space behavior.
  • normal
  • nowrap
  • pre
  • pre-line
  • pre-wrap
line-height The line-height property specifies the height of an element.
  • normal
  • length
  • %
  • Numeric value
letter-spacing The letter-spacing is used to sets the horizontal spacing behavior between text characters. This value is added to the natural spacing between characters while rendering the text
  • normal
  • length
word-spacing The word-spacing property increases or decreases the white space between words
  • Numeric value
  • normal
  • length
text-indent The text-indent property is used to indent the first line of text in an element.
  • length
  • %
text-align The text-align property is used to specify the horizontal alignment of text within an element.
  • left
  • right
  • center
  • justify

LIST PROPERTIES

The list properties are used to specified the type of the item marker, such as ordered type or unordered type. It detemines the apperance of the list item marker of a list element.

LIST PROPERTIES AND DESCRIPTION
LIST PROPERTY DESCRIPTION VALUE(s)
list-style-image It is used to set an image as the marker for list items (such as bullets or numbers) or ordered list. Instead of using the default markers provided by the browser.
  • URL
  • none
  • inherit
list-style-position It is used to specify the position of the markers in a list relative to the list item's content.
  • inside
  • outside
  • inherit
list-style-type It is used to specify the type of marker used for list-items in ordered or unordered lists. It determines the appearance of the marker before each item.
  • none
  • disc
  • circle
  • decimal
  • decimal-leading-zero
  • armenian
  • georgian
  • upper-roman. e.t.c
list-style It sets all list-properties for a list in one declaration. It combines the list-style-image, list-style-position, list-style-type properties into a single line.
  • list-style-image, list-style-position, list-style-type

DIMENSION PROPETIES

Dimension properties refer to a set of attributes that control the spatial charateristics of HTML elements on a webpage.

DIMENSION PROPERTIES
DIMENSION PROPERTY DESCRIPTION VALUE(s)
width The width property is used to specify the horizontal size of an element. It determines how wide an element should be rendered within its containing parent element or the viewport. The number value assigned to the width property can be specified using various units. UNITS
  • pixels (px)
  • percentages(%)
  • viewport(vw)
  • em
  • rem
height The height property is used to specify the vertical size of an element. It determines how tall/long an element should be rendered within its containing parent element or the viewport. The number value assigned to the height property can be specified using various units. UNITS
  • pixels (px)
  • percentages(%)
  • viewport(vw)
  • em
  • rem
max-width The max-width property sets the maximum width of an element. It specifies the maximum allowable width an element can have, preventing it from expanding beyond that limit. max-width property can be specified using various units. UNITS
  • pixels (px)
  • percentages(%)
  • viewport(vw)
  • em
  • rem
min-width The min-width It specifies the smallest allowable width an element can have, ensuring that it does not shrink below that limit. The min-width property can be specified using various units. UNITS
  • pixels (px)
  • percentages(%)
  • viewport(vw)
  • em
  • rem
max-height The max-height property sets the maximum height of an element. It specifies the maximum allowable height an element can have, preventing it from expanding beyond that limit. max-height property can be specified using various units. UNITS
  • pixels (px)
  • percentages(%)
  • viewport(vh)
  • em
  • rem
min-height The min-height It specifies the short allowable height an element can have, ensuring that it does not shrink below that limit. The min-height property can be specified using various units. UNITS
  • pixels (px)
  • percentages(%)
  • viewport(vh)
  • em
  • rem
