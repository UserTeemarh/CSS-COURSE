TRANSFORM PROPERTY

The transform property is used to apply transformations to elements. The transform property is used to modify the apperance and position of an element without affecting the layout of the document.

All transform values are to be specified in number.

TYPES OF TRANSFORM PROPERTY'S VALUES

TRANSFORM PROPERTY'S VALUES AND DESCRIPTION
TRANSFORM PROPERTY'S VALUES DESCRIPTION UNIT
translateX( ) The translateX( ) value is used to move an element along the horizontal (X) axis. It shifts the element horizontally by specified distance px
translateY( ) The translateY( ) value is used to move an element along the vertical (Y) axis. It shifts the element vertically by specified distance px
translate( ) The translate( ) value is used to within the transform property to move an element along with the horizontal (X) and vertical (Y) axes simultaneously. It specifies the amount by which the element should be translated in the X and Y directions px
translateZ( ) The translateZ( ) is used move an element along the z-axis in 3D space. It specifies an element to be translated closer or or farther away from the viewer, creating a sense of depth or persceptive. px
rotateX( ) The rotateX( ) value is used to rotate an element around the horizontal (X) axis in a 3D space. It specifies the angle by which the element should be rotated along the X axis. deg
rotateY( ) The rotateY( ) value is used to rotate an element around the vertical (Y) axis in a 3D space. It specifies the angle by which the element should be rotated along the Y axis. deg
rotate( ) The rotate( ) value is used to rotate an element by a specified angle around its center point. deg
rotateZ( ) The rotateZ value is used to rotate an element around the z-axis in a 2D or 3D. It specifies the angle by which the element should be rotated along the z-axis. deg
scaleX( ) The scaleX( ) is used to scale an element along the horizontal (X) axis. It specifies the scaling factor by which the element should be resized along the x-axis. %
scaleY( ) The scaleY( ) is used to scale an element along the horizontal (Y) axis. It specifies the scaling factor by which the element should be resized along the Y-axis. %
scale( ) The scale( ) value is used to scale an element both horizontally and vertically. It specifies the scaling factors along both axes. %
scaleZ( ) The scaleZ( ) value is used to to scale an element along the z-axis in 3D space. It specifies the scaling factor by which the element should be resized along the z-axis %
skewX( ) The skewX( ) value is used to tilt or skew an element along the horizontal (X) axis. It specifies the angle by which the element should be skewed along the x-axis deg
skewY( ) The skewY( ) value is used to tilt or skew an element along the vertical (Y) axis. It specifies the angle by which the element should be skewed along the Y-axis deg
skew( ) The skew( ) value is used to tilt an element along both the horizontal (X) and vertical (Y) axes simultaneously. It specifies the angles by which the element should be skewed along both axes deg
matrix( ) The matrix( ) value is a powerful tool that specifies a 2D transformation using 3x3 matrix. It provides more control over transformations compared to individual transformation values. no unit
It is used to combine all transforms into one and perform complex transformation by specifying the values of transformation matrix. The matrix() takes six parameters.
matrix( ) PARAMETERS
scaleX() Horizontal scaling: It's the value by which the width of the element is scaled. A value of 1 means the width is unchanged.
skewY( ) Skew transformation along the X-axis. It's the angle (in degrees) by which the element is skewed along the Y-axis
skewX( ) Skew transformation along the X-axis. It's the angle (in degrees) by which the element is skewed along the x-axis.
scaleY( ) Vertical scaling: It's the value by which the height of the element is scaled. A value of 1 means the height is unchanged.
translateX( ) Horizontal translation. It's the distance by which the element is moved along the X-axis. Positive values will move the element to the right, while negative values will move it to the left.
translateY( ) Vertical translation: It's the distance by which the element is moved along the Y-axis. Positive values will move the element down, while negative values will move it up.
