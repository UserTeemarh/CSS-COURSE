FLOAT AND DISPLAY TYPE PROPERTY

FLOAT PROPERTY

The flaot property is used for positioning and formatting content. It is used to specify whether an element should to the left or right of its container, allowing other elements to flow around it.

FLOAT PROPERTY VALUES
VALUE DESCRIPTION
left: Makes the content float on the left side of the container.
right: Makes the content float on the right side of the container.
none Makes the content not to float
inline-start Makes the content float on the start side of its container
inline-start Makes the content float on the end side of its container

DISPLAY PROPERTY

The display property is used to specify how an element should be rendered in the document. It controls the type of box used for an HTML element.

DISPLAY PROPERTY VALUES
DISPLAY VALUE: DESCRIPTION
block: It generates a block-level box, which makes it takes up the entire width available to it and starts a new line.
inline: It generates an inline-level box, which means it flows within the text on the same line as adjacent content.
inline-block It generates an inline-level box, but it behaves like a block-level element in terms of layout.
flex: It generates a block-level flex container, enabling flexible layout options for its children.
grid: It generates a block-level grid container, enabling grid layout options for its children.
table: It makes an element to behave like a table element.
table-caption It makes an element behave like a table caption element, typically used to provide a title or description for a table.
inline-table: It makes an element behave like an inline-level table element.
table-row It makes an element behave like an table row element
table-cell It makes an element behave like a table cell element.
none It removes the element fro the document flow and does not take up any space on the page.
