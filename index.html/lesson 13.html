THE TRANSITION PROPERTIES

Transition property allows you to change property values smoothly over a given duration.

Transition-property
The transition property specifies the name of the CSS property the transition effect is for. (The transition effect will start when the specified CSS property changes).
Transition-duration
This property specifies the amount of time a transition effect should take to complete, and the duration can be adjusted by changing the value to your preffered time.
Transition-timing-function
This property specifies the speed curve of the transition effect. It allows a transition effect to change speed over its duration.

Transition-timing function has four (4) values:

  • ease:(default) slow at the beginning, fast in the middle, slow at the end
  • ease-in: starts slowly and accelerate
  • ease-out: starts quickly and decelerates
  • ease-in-out: starts slowly, accelerates in the middle, and decelerates at the end.
Transition-delay
This property specifies the duration to wait before starting a property's transition effect when its value changes.
